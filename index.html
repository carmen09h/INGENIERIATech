  3. Sistema

REALIDAD VIRTUAL

Tecnología que crea un entorno simulado, en el cual los usuarios pueden interactuar de manera inmersiva usando dispositivos especiales como cascos VR y controladores.

TIPOS

* No Inmersiva

* Semi-Inmersiva

* Totalmente Inmersiva

ELEMENTOS

* DISPOSITIVOS DE VISUALIZACIÓN:Cascos VR.

* SENSORES DE MOVIMIENTO: Controladores y guantes.

* SOFTWARE DE REALIDAD VIRTUAL:Programas que crean los entornos virtuales.

ROBOTICA

Rama de la ingeniería y ciencia que incluye la concepción, diseño, fabricación y operación de robots. La robótica implica el uso de sistemas computarizados para controlar y programar estos dispositivos.

TIPOS

* Industriales

* Servicio

* Moviles

ELEMENTOS

* ACTUADORES:Los cuales permiten el movimiento y funcionamiento.

* SENSORES: Se encargan de capturar infromación del entorno.

* CONTROLADORES: Encargados de procesar la información y envian señales a los actuadores.

* SOFTWARE: Programas que controlan el comportamientodel robot.

BIG DATA

Conjuntos de datos extremadamente grandes y complejos que requieren tecnologías avanzadas para su procesamiento, análisis y gestión.

TIPOS

* Datos Estructurados: Datos organizados en tablas, como bases de datos.

* Datos No Estructurados: Información sin formato específico, como correos electrónicos.

* Datos Semiestructurados: XML, JSON.

ELEMENTOS

* VOLUMEN:Cantidad de datos.

* VARIEDAD: Diferentes tipos de datos.

* VELOCIDAD: Rapidez con que se generan y procesan los datos.

* VERACIDAD: Calidad y precisión de los datos.

* VALOR: Importancia de los datos

IoT

El Internet de las Cosas (IoT), es la interconexión de dispositivos a través de Internet, permitiéndoles enviar y recibir datos.

TIPOS

* Dispositivos de Consumo: Hogares inteligentes, wearables.

* Dispositivos Industriales: Maquinaria y sensores en fábricas.

* Infraestructura Urbana: Ciudades inteligentes.

ELEMENTOS

* SENSORES Y ACTUADORES: Recogen datos y realizan acciones.

* CONECTIVIDAD: Redes que conectan los dispositivos.

* PLATAFORMAS DE GESTIÓN: Software para analizar y gestionar datos.

* INTERFAZ DE USUARIO: Aplicaciones para el control y monitoreo.

________________________________________________________________________________________________________________

REDES

Interconexión de dispositivos para compartir recursos y comunicarse entre sí.

TIPOS

* Redes de Área Local (LAN): Redes en un área geográfica limitada.

* Redes de Área Metropolitana (MAN): Conexiones en una ciudad.

* Redes de Área Amplia (WAN): Conexiones a nivel nacional o internacional.

Redes Privadas Virtuales (VPN): Extienden redes privadas a través de redes públicas.

ELEMENTOS

* DISPOSITIVOS DE RED: Routers, switches, hubs.

* MEDIOS DE TRANSMICIÓN: Cables de cobre, fibra óptica, inalámbrico.

* PROTOCOLOS DE COMUNICACIÓN: TCP/IP, HTTP, FTP.

* SOFTWARE DE GESTIÓN: Herramientas para monitorear y gestionar redes.

CIBERSEGURIDAD

Protección de sistemas informáticos, redes y datos contra ataques, daños o acceso no autorizado.

TIPOS

* Seguridad Operacional: Protección de procesos y operaciones.

* Seguridad de Redes: Protección de redes informáticas.

* Seguridad de Aplicaciones: Protección de software y aplicaciones.

* Seguridad de Información: Protección de datos.

* Seguridad Operacional: Protección de procesos y operaciones.

ELEMENTOS

* FIREWALL: Barreras de seguridad.

* ANTIVIRUS Y ANTIMALWARE: Software de protección.

* CIFRADO: Protección de datos.

* AUTENTICACIÓN Y AUTORIZACIÓN: Control de acceso.

MACHINE LEARNING

El aprendizaje automático es una rama de la inteligencia artificial que permite a los sistemas aprender y mejorar automáticamente a partir de la experiencia sin ser programados explícitamente

TIPOS

* Aprendizaje Supervisado: Modelos entrenados con datos etiquetados.

* Aprendizaje No Supervisado: Modelos que encuentran patrones en datos sin etiquetas.

* Aprendizaje por Refuerzo: Modelos que aprenden mediante recompensas y castigos.

ELEMENTOS

* CUNJUNTO DE DATOS: Datos para entrenar los modelos..

* ALGORITMOS DE APRENDIZAJE: Técnicas para construir modelos predictivos.

* MODELOS PREDICTIVO: Resultados del entrenamiento con los dato.

* VALIDACIÓN Y EVALUACIÓN: Métodos para comprobar la precisión de los modelos.

________________________________________________________________________________________________________________

INTELIGENCIA ARTIFICIAL

La inteligencia artificial se refiere a la simulación de procesos de inteligencia humana por parte de sistemas informáticos, incluyendo aprendizaje, razonamiento y autocorrección.

TIPOS

* IA Débil: Sistemas diseñados para tareas específicas, como asistentes virtuales.

* IA Fuerte: Sistemas con capacidad de razonamiento y cognición similar a la humana.

ELEMENTOS

* ALGORITMOS DE APRENDIZAJE:Métodos para aprender de datos.

* REDES NEURONALES: Modelos inspirados en el cerebro humano.

* PROCESAMIENTO DE LENGUAJE NATURAL: Interacción con el lenguaje humano.

* SISTEMAS DE PERCEPCIÓN: Visión por computadora, reconocimiento de voz.
